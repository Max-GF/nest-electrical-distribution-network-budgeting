# nest-electrical-distribution-network-budgeting
Projeto em Nest para um sistema para orçamento de materiais em projetos de redes elétricas de distribuição.

## Instalação

1. Clone o repositório:
\`\`\`bash
git clone https://github.com/seu-usuario/nest-electrical-distribution-network-budgeting.git
\`\`\`

2. Entre na pasta do projeto:
\`\`\`bash
cd nest-electrical-distribution-network-budgeting
\`\`\`

3. Instale as dependências:
\`\`\`bash
pnpm install
\`\`\`

## Executando a aplicação

\`\`\`bash
# desenvolvimento
pnpm run start

# modo watch
pnpm run start:dev

# modo produção
pnpm run start:prod
\`\`\`

## Testes

\`\`\`bash
# testes unitários
pnpm run test

# testes e2e
pnpm run test:e2e

# cobertura de testes
pnpm run test:cov
\`\`\`
